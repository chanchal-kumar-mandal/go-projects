# Golang RESTful API using GORM and Gorilla Mux

## Requirement
  
  go get -u github.com/gorilla/mux
  go get -u github.com/jinzhu/gorm
  go get -u github.com/go-sql-driver/mysql
  
  Postman to test API

## Details

   * Show employees
   * Create employee 
   * Update employee
   * Delete employee
